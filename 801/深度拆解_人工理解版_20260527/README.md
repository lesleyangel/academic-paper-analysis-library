这套目录用于替代此前的自动填表版。核心原则是：先理解论文如何完成学术说服，再把所有内容归纳进十层框架；十层不是报告中的一个部分，而是整篇报告的唯一主结构。

## 执行标准

每篇论文报告只使用以下十个一级标题：

```text
1. 身份层
2. 主张层
3. 选题层
4. 位置层
5. 贡献层
6. 方法层
7. 证据层
8. 结构语言层
9. 引用风险层
10. 复用层
```

摘要、结论、逐段理解、图表、公式、Claim-Evidence、词频句式、引用策略、审稿风险、可模仿写法、不可模仿内容，全部收进对应层级，不另开并列大框架。

## 使用说明

阅读单篇时，直接进入 `papers/` 查找对应编号。做团队画像或横向归纳时，优先使用每篇的：

- `主张层`：提取核心 claim 和摘要/结论闭合方式。
- `位置层`：提取文献版图和 gap 类型。
- `贡献层`：提取真实贡献和创新强度。
- `证据层`：提取图表、公式、实验、Claim-Evidence。
- `结构语言层`：提取章节推进、段落功能和语言习惯。
- `引用风险层`：提取审稿风险和引用策略。
- `复用层`：提取可迁移写法和不可模仿边界。
