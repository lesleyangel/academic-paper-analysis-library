这是一篇复合材料有效热导率预测中的单胞构造规则论文。它与第 043 篇热膨胀单胞论文相互关联，但本篇关注的是热传导问题中的温度分布规则和热边界条件，而不是热弹性位移边界条件。它的真实身份可以概括为：**面向对称结构复合材料热计算的多尺寸单胞公式化方法论文**。

研究对象是具有平移、反射和 180° 旋转对称性的宏观/细观复合材料，包括 UD 纤维增强复合材料和 4HS-8HS 缎纹织物复合材料。方法组合包括 Fourier 热传导、宏观热流方向定义、对称/反对称热刺激、温度分布规律、单胞几何缩减、边界条件推导和有限元有效热导率计算。

## 2. 主张层

一句话主张是：**具有结构对称性的复合材料有效热性能可以用按对称性缩减的单胞高效计算，但每次利用平移、反射或 180° 旋转对称性缩小几何时，都必须根据宏观热流方向对应的对称/反对称温度分布规律重新推导热边界条件。**

摘要完整意译为：宏观对称结构复合材料的有效热性能可由尺寸有限的代表性单胞高效计算；本文发展一般单胞构造规则，包括识别结构对称性和推导边界条件两步，前者建立几何模型，后者赋予模型代表原结构的物理意义；从实验试样到单胞模型的路线，尤其是边界条件演化，被阐明；宏观热流按方向定义为对称热刺激和反对称热刺激；对平移、反射和 180° 旋转三类典型对称结构，总结温度分布规律为两类方程，用于推导单胞边界条件；UD 轴向算例展示了正确推导边界条件与不恰当边界条件的物理差异；横向 UD 建立四个缩减单胞；缎纹复合材料分析 4HS-8HS 结构对称性，建立两个单胞并统一推导边界条件；多尺寸单胞一致结果验证了方法，可推广到其他具有相关对称性的复合材料热研究。

结论完整意译为：本文建立了对称结构复合材料热计算的单胞构造通用规则，规则基于宏观实验试样和测量条件；宏观热流按方向被定义为 STS 和 ATS；不同对称结构下的温度分布规律用方程描述并由典型计算确认，可用于推导单胞边界条件；UD 轴向研究证明推导边界条件必要性，横向研究展示单胞构造与边界条件推导流程，缎纹复合材料中两个尺寸单胞得到一致结果并与文献实验相符；结论强调四点：对称复合材料可用单胞高效计算；基本流程是识别对称性、建几何、确定宏观热流方向、推导 BC；每次利用对称性都会产生更小单胞但需要新 BC；缎纹综数增加时面内热导略增、面外热导略降。

## 3. 选题层

选题来源于有效热性能预测中的“代表性与成本”矛盾。复合材料在高速飞行器等严酷热环境中服役，必须先获得有效热导率等热特性；全尺寸细观建模成本高，因此通常选择 RVE/单胞。但是单胞只有在几何和边界条件都代表原结构时才有意义。

对于随机结构，RVE 与边界条件通常近似；而对于有序结构，结构对称性提供了构造精确单胞的机会。问题是：平移周期边界大家熟悉，但反射和 180° 旋转对称如何改变温度边界条件并不清楚。若只追求几何缩小而不重新推导热边界条件，就可能得到计算上很快但物理上错误的模型。

本文把选题压缩为：如何从宏观热流方向出发，判断对称结构中的温度场是 STS 还是 ATS，并据此推导缩减单胞边界条件？研究边界是稳态/有效热导率热传导计算，不涉及热膨胀、损伤、非线性辐射或随机缺陷。

## 4. 位置层

文献版图有三条线。第一条是复杂多相材料有效热导率预测，包括随机多孔、泡沫、颗粒、燃料电池扩散层等 RVE 方法，说明 RVE 是通用需求但随机结构只能近似代表。第二条是纺织增强复合材料热导率单胞研究，已有平纹、缎纹、三维编织、针刺等模型，多用完整平移周期单胞。第三条是多尺寸单胞和结构对称性利用，已有作者前期关于平纹和三维四向编织复合材料的多尺寸单胞研究，但尚缺少可推广到不同对称性的热物理规则总结。

Introduction 的 gap 是：已有研究普遍先构造完整周期单胞，少量工作利用反射/旋转缩减尺寸，但缺少一般规则、温度分布物理机制和从实验热流方向到单胞 BC 的系统说明。本文接在周期单胞和作者前期多尺寸单胞之后，试图把“会用对称性”提升为“可推导规则”。

这个 gap 成立。最接近工作是 Li 关于机械单胞、作者关于平纹/四向编织有效热导率的多尺寸单胞，以及常规周期热边界条件研究。本文的新位置是热计算中 STS/ATS 的抽象和统一边界条件推导。

## 5. 贡献层

作者声明的贡献包括：定义宏观热流方向下的 STS/ATS；总结三类对称结构中的温度分布规律；展示从实验试样到单胞模型的边界条件演化；用 UD 和缎纹复合材料验证多尺寸单胞；给出缎纹综数对面内/面外热导率的趋势。

真实贡献有三层。第一是概念贡献，用 STS 和 ATS 把宏观热流方向与局部温度对称关系连接起来。第二是方法贡献，给出四步流程：识别对称性、建立几何、确定 q0 方向、推导边界条件。第三是验证贡献，通过 UD 轴向错误边界案例、多尺寸 UD 横向单胞和 4HS-8HS 缎纹单胞一致性证明边界条件不是任意选择。

创新强度属于方法规则型中等偏强。它不是新求解器，也不是新材料；价值在于把边界条件这个容易被经验化处理的部分规则化。贡献边界是：假设结构理想对称、材料参数给定、热传导线性且宏观热流清晰；对实际纤维波动、孔隙、界面热阻和三维复杂缺陷没有展开。

## 6. 方法层

方法从 Fourier 宏观热传导本构出发，将宏观热流 q0 分解到 x、y、z 方向。随后作者根据宏观热流与对称面/对称轴方向的相对关系，定义对称热刺激 STS 与反对称热刺激 ATS：在 STS 下对称点相对于基准温度的温差相等，在 ATS 下温差符号相反。Eq. (3) 和 Eq. (4) 是整个方法的核心。

流程可以写成：

```text
宏观试样和热流方向 -> 识别平移/反射/180°旋转对称性
-> 判断该方向属于 STS 还是 ATS
-> 用温度分布规律推导边界温度或绝热条件
-> 构造缩减单胞 -> 有限元计算有效热导率
-> 多尺寸单胞结果一致性检验
```

UD 轴向算例用于说明不恰当 BC 的问题：如果只施加局部不符合宏观热流分布的边界条件，会在边界对称节点温差上产生错误。UD 横向算例则展示四个逐步缩小的单胞 UC1-UC4，Table 7 中 λxx 与 λyy 约 1.6503-1.6506 W/(m·K)，说明多尺寸一致。缎纹算例中 4HS-8HS 的面内热导约 2.037-2.049 W/(m·K)，面外约 0.5439-0.5580 W/(m·K)，两个单胞偏差很小。
