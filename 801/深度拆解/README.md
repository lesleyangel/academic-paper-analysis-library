## 使用建议

先读 `batch_summaries/` 建立主题地图，再进入 `papers/` 查单篇。若目标是模仿写法，优先看每篇 `## 11` 到 `## 15` 中并入的章节、正文词频、句型和引用复核；若目标是找选题与创新套路，优先看 `## 3` 到 `## 8`；若目标是训练审稿视角，优先看 `## 9`、`## 10`、`## 16`。

## 注意事项

本批拆解主要基于 PDF 文本抽取。对云图、显微图、实验照片、表格精确列名、复杂公式排版和图中曲线细节，单篇文件中已标注“需要 PDF 图像复核”。这些标注不是错误，而是提醒后续如果要引用精确数值或复刻图表，应回到原 PDF 做视觉核对。
