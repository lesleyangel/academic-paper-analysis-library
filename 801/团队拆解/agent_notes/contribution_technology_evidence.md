# 801 团队拆解笔记：贡献栈、技术工具箱与证据风格

## 0. 依据与总判断

样本依据：`801/深度拆解/papers/*.md` 共 49 篇，`801/深度拆解/batch_summaries/*.md` 共 9 个批次总结，并用 `801/文本/_manifest.json` 核对题名、期刊、年份和关键词。样本范围主要覆盖 2016-2026 年，集中在 Aerospace Science and Technology、Applied Thermal Engineering、Composite Structures、International Journal of Heat and Mass Transfer、Probabilistic Engineering Mechanics、Chinese Journal of Aeronautics 等工程期刊。

总体判断：801 论文集合的稳定生产方式不是单一“理论突破”，而是把高超声速/航空航天复杂系统中的高成本仿真、强耦合热-力-电-气动问题、有限传感器观测和多尺度材料结构，转化为可计算的中间表示，再用代理模型、降阶模型、优化算法、等效模型或实验校准把它们推回工程指标。这个判断为强证据，跨热管理、热电、ROM、可靠性、优化、单胞/多尺度和气动控制多个批次反复出现。

## 2. 资产复用链

### 2.1 反复使用的理论与模型

- 强证据：热传导、热平衡、CHT/CST、等效换热系数 `heq`、热阻/电阻等效层，是热管理、热防护、热电和温度反演的底层资产。2019 TMS、2021 heat dissipation/reuse、2025 heat flux POD+TSCN、2025 POD-BPNN-GA、2026 TFNO 等都围绕热场/热流建模。
- 强证据：POD/ROM/模态系数是高维场压缩的长期资产。热流预测、温度反演、流场重构、气动载荷 DMS-LSTM、FSI flow sensing 都使用“高维场 -> 低维系数/模态 -> 学习或优化 -> 场重构”的链条。
- 强证据：代理模型与机器学习反复复用，包括 Kriging、RBF/RBFNN、BPNN、DNN、LSTM、TCN/CNN、TFNO、KRR、随机森林 importance、transfer learning。它们通常不是孤立算法，而是嵌入热/气动/结构/传感流程。
- 中证据：单胞、最小尺寸 unit cell、对称边界条件、Eshelby-Mori-Tanaka、多尺度 FE、Nataf 相关传播，构成复合材料与多尺度 UQ 支线的稳定资产。
- 中证据：可靠性与不确定性模型反复出现，包括 MCS、FORM/SORM、AK-MCS、RBF sequential surrogate、BMPPT、EOLE、Chebyshev/P-box、Nataf/maximum entropy。它们服务于“少调用昂贵极限状态函数/高保真模型”的共同目标。

### 2.2 反复使用的实验平台、数据与工程案例

- 强证据：高保真数值平台是最主要数据来源，包括 CFD、FEM、CHT/CST、CFD/RBD、FSI、COMSOL 热电多物理场、Nastran/Fluent 等。很多论文的强证据来自高保真基准和代理/降阶模型对比。
- 中证据：热电与热接触实验平台复用明显，包括粗糙面测量、TCR/ECR 间接测试、TEG 输出电压实验、多层 TEG/RTG 性能验证。ECR、TCR/NFTR、2025 气动热采集 TEG、2025 火星 RTG 多层 TEG 形成连续链条。
- 中证据：材料实验资产包括热电陶瓷 XRD/SEM/XPS/Raman/TEM/EBSD/热电性能测试，以及 FRP 高温压缩、太阳辐照热响应、微结构/孔隙统计。它们强在材料方向，团队整体复用度中等。
- 强证据：工程案例反复围绕高超声速/再入/高速飞行器：再入胶囊、再入舱、可重复使用运载器、巡航飞行器、X37-like vehicle、高速翼、舵结构、进气道外压缩段、双旋弹丸。MAV、植被 FSI、微重力细胞属于支线或合作型扩展。

### 2.3 反复使用的指标与图表结构

- 强证据：精度 + 效率双指标是最稳定指标组合。常见指标包括 MRE/mMRE/MAE/RMSE/R2/NSE、相对误差、局部峰值误差、CPU time、仿真次数、LSF evaluations、NFE/NBMPP、训练/推理时间。
- 强证据：工程性能指标高度稳定，包括 TPS 重量、冷却剂质量流量、输出功率、转换效率、GPD、ZT/PF、payload mass、结构质量、温度/应力/位移约束、终端误差、成功率、传感器数量。
- 强证据：图表结构复用稳定：方法流程图、工程对象几何图、云图对比、误差云图、收敛曲线、参数敏感性曲线、baseline vs proposed 表、多工况统计表、输入-输出相关矩阵、传感器布局图、机制示意图。
- 中证据：代码/工具复用只能间接判断。深拆显示大量自建流程和算法，但现有材料不足以判断是否有公开代码或跨论文直接复用同一代码库。

## 3. 技术工具箱

### 3.1 核心技术

- 强证据：高保真数值建模是底座。CFD/FEM/CHT/CST/FSI/RBD/多物理场仿真承担标签生成、机制解释和 baseline 功能。
- 强证据：降阶与代理模型是最核心方法族。POD、非线性 ROM、KRR-DCR、DMS、POD-BPNN、POD-RBFNN、TFNO、LSTM/DNN/TSCN 等反复用于替代昂贵场求解。
- 强证据：优化与可靠性算法是第二核心方法族。GA、hybrid GA、RBF 预筛选、SCP/hp-adaptive RPD、NMPG/ET-NMPG、MDO/ALC、RBF/Kriging reliability、P-box/UP 被用于把模型放进反复设计或安全评估循环。
- 强证据：热-电-力多物理建模是热防护/热电方向核心。TCR/ECR、NFTR、等效界面层、TEG/RTG 系统模型、active/regenerative cooling 与 TPS 分区共同形成一条长链。
- 中证据：多尺度/单胞/随机孔隙传播是复合材料方向核心，在 2017-2018 unit cell、2016 FRP、2025 woven composite UQ、2026 stochastic porosity stiffness propagation 中稳定出现。

### 3.2 辅助技术

- 强证据：参数扫描、敏感性分析、消融对比、网格/时间步独立性、文献或软件对比，是工程方法论文的标准辅助工具。
- 中证据：实验验证常作为局部关键环节使用，如 TCR/ECR、TEG 输出、MAV 实飞、太阳辐照热响应、材料性能测试。但在多数高超声速系统级论文中，实验不是主证据。
- 中证据：随机采样和统计工具包括 LHS、Monte Carlo、Nataf transformation、maximum entropy、Gaussian process、P-box、DBA/dropout uncertainty，主要用于不确定性、可靠性和代理模型可信度。

### 3.3 新引入技术与复杂度变化

- 强证据：技术复杂度从 2016-2019 的等效模型、单胞边界、实验装置、传统 FE/CFD，演化到 2024-2026 的神经算子、多源传感、低保真融合、DMS-LSTM、WAPE-assisted ET-NMPG、多学科 UP/MDO 和 inter-scale stochastic propagation。
- 中证据：深度学习工具近年明显增加，但多数论文把新意放在物理表示或部署约束上，而不是宣称网络结构本身原创。TFNO、TSCN、DMS-LSTM、DNN transfer learning、BPNN/GA 的共同写法是“物理中间变量 + 学习器”。
- 中证据：材料方向的新技术包括高熵调控、GO 多机制协同、织构/缺陷/界面协同、还原退火诱导 n 型行为。它们为热电性能提供材料侧补充，但与飞行器系统工具箱的耦合程度还有限。

### 3.4 技术组合方式

- 强证据：典型组合为“工程高成本问题 -> 中间表示 -> 代理/降阶/优化 -> 高保真或实验验证 -> 工程指标”。例如热流 POD+TSCN、TFNO sparse sensors、DMS-LSTM、temperature inversion、MDO 和热-力路径优化都符合这一结构。
- 强证据：另一稳定组合为“微观/界面机制 -> 等效层/等效参数 -> 器件或系统模型 -> 输出功率/效率/重量”。TEG/TCR/ECR/NFTR/RTG 系列非常典型。
- 中证据：可靠性/UQ 方向组合为“随机输入/不完整分布 -> 低成本传播或代理极限状态 -> 与 MCS/FORM/SORM/高保真比较 -> 误差与函数调用数”。这是方法论文的固定产线。

## 5. 证据风格

### 5.1 常见图表与证据结构

- 强证据：图驱动叙事非常稳定。结果段常按“Fig./Table shows -> trend -> reason -> design implication”推进，云图和曲线承担主要解释，表格承担最终 claim。
- 强证据：常见图表包括工程构型图、方法流程图、网络/算法结构图、温度/热流/压力/应力/位移云图、误差云图、涡结构/Q-criterion、传感器布局、收敛曲线、参数扫描曲线、baseline 对比表。
- 强证据：复杂论文喜欢把证据分层：基准验证 -> 参数/消融 -> 工程案例 -> 效率或系统指标。例如 TFNO、TSCN、DMS-LSTM、SCP、MDO、TEG/RTG 都采用多层证据链。

### 5.2 Baseline 与对比习惯

- 强证据：baseline 覆盖充分。方法论文常与直接高保真仿真、传统 ROM、POD、RBF/Kriging/BPNN/CNN/FNO/UNO、MCS/FORM/SORM/AK-MCS、GPOPS-II、PID/LQR、force-only 设计、理想无接触模型等比较。
- 中证据：对比公平性有时依赖作者调参深度。深拆中多次指出网络/代理/GA/SCP 的超参数、随机种子、多次运行统计仍可能影响结论。

### 5.3 误差、敏感性、消融与机制解释

- 强证据：方法论文普遍重视误差和效率并报。典型指标包括相对误差、RMSE、MAE、CPU time、函数调用数、训练/推理时间、传感器减少比例、输出功率/效率、质量降低比例。
- 强证据：近年论文明显增加消融与敏感性，如 TSCN 输入消融、TFNO 输入组合和低保真质量敏感性、POD-BPNN-GA 降维/网络/传感器调参、DMS-LSTM 表示消融、RBF 预筛选和层级变量对比。
- 中证据：机制解释总体较强，但材料多机制与深度学习模型仍有解释风险。热电材料常有多因素同时变化，ML 模型常需要通过模态、传感器位置或 feature importance 补充解释。

### 5.4 可复现性风格

- 中证据：论文通常给出较多工程设置、工况、材料参数、训练样本、优化变量和评价指标，支持读者理解方法。
- 中证据：严格复现仍受限于 CFD/FEM 数据、网格/边界条件、代码、随机种子、实验装配、PDF 图表细节。深拆中大量标注“需要 PDF 图像复核”，说明文本抽取不足以完全锁定图形证据。

## 6. 证据短板与审稿风险

1. 强证据：系统级论文过度依赖仿真，真实风洞、飞行、长期服役、热循环和装配退化验证不足。热防护、TEG/RTG、再入制导、损伤识别、DMS-LSTM、MDO 都存在不同形式的外推风险。
2. 强证据：训练/验证数据域有限。15 条轨迹、单一再入舱、单一翼/舵/车辆、固定 Mach/AOA/高度组合、单一材料体系或固定损伤形态，可能支撑插值但不足以支撑广泛外推。
3. 强证据：平均误差可能掩盖关键局部风险。热流峰值、热点位置、局部应力集中、接触界面失效、传感器遮蔽区、路径约束边界等需要峰值/局部/安全裕度指标。
4. 中证据：组合式创新容易被质疑“算法组件不新”。TSCN、TFNO、DMS-LSTM、BPNN-GA、event-triggered NMPG、MDO/ALC 等都需要靠工程嵌入位置、消融和下游指标稳住贡献。
5. 中证据：baseline 公平性和随机优化稳定性是常见风险。GA、RBF 筛选、神经网络、Kriging、SCP/NMPG 阈值对初值、随机种子、超参数和训练预算敏感。
6. 中证据：材料和界面机制归因存在耦合风险。高熵、GO、氧空位、孔隙、rGO 网络、第二相、界面层、银胶、氧化膜等因素常同时变化，单因子证据不足时容易被材料审稿人追问。
7. 中证据：工程部署约束经常后置。传感器耐温/布线、冷却通道制造、热电界面长期稳定、复杂热-力路径制造、机载算力和数据延迟，在现有论文中多为未来工作。
8. 弱证据：跨方向支线可能稀释团队画像。微重力转录组、植被 FSI、MAV 控制等论文各自证据完整，但在 801 主贡献栈中更像合作或外延方向，不能据此判断团队主资产发生转向。

## 7. 可迁移的贡献栈模板

- 强证据：最可复用的论文生产模板是“找到高保真流程中最昂贵或最不可信的环节，建立物理上可解释的中间表示，用代理/降阶/优化替代它，再用精度、效率、敏感性和工程指标证明可用”。
- 强证据：第二个模板是“把被当作误差项的非理想因素提升为主贡献”，如 TCR/ECR/NFTR、壁温非均匀性、历史飞行状态、传感器稀疏性、损伤容限、接触界面、变量层级。
- 中证据：第三个模板是“用尺度递进完成系统化贡献”，即微观/界面/单胞 -> 等效参数/低维表示 -> 器件/结构/整机 -> 工程性能。
- 中证据：最值得总报告吸收的表达套路是 Claim-Evidence 对齐：每个方法模块对应一个失败模式，每个强 claim 至少配一个量化表、一个可视化图和一个 baseline。
