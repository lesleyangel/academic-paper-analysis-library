# 学术期刊蒸馏器

这是一个论文拆解与期刊写作分析库，用于沉淀论文选题、论证链、创新性、图表叙事、章节布局、文笔习惯、引用策略和审稿风险等分析结果。

## 内容

- `论文拆解框架与模板.md`：通用论文深度拆解框架。
- `期刊拆解框架与模板.md`：期刊层面的风格与选题拆解框架。
- `个人团队拆解分析模板.md`：面向个人或团队的复用模板。
- `jmps/深度拆解/`：JMPS 论文深度拆解结果。
- `jmps/拆解/`：JMPS 早期标准拆解结果，可作为轻量版参考。
- `801/深度拆解/`：801 论文集合深度拆解结果。
- `aei/_extracted/agent*.md`：AEI 论文写法与主题聚类分析笔记。
- `tools/`：PDF 文本抽取、质量检查、语言/结构分析增强脚本。

## 不包含的内容

公开仓库不包含原始论文 PDF、ScienceDirect 下载包和全文抽取文本。它们通常受出版方版权约束，因此通过 `.gitignore` 排除。仓库保留的是基于论文阅读形成的拆解分析、模板、批次总结和辅助脚本。

## 主要拆解维度

- 论文身份与研究定位
- 摘要和核心信息提取
- 选题来源与问题收束
- 文献版图与 gap 制造机制
- 创新性判断
- 论证链条和方法模型复原
- Claim-Evidence 对照
- 图表、公式和结果叙事
- 章节结构、段落功能和篇章布局
- 文笔画像、常用词、句式、主动/被动、时态和学术词汇
- 引用策略与文献使用偏好
- 审稿人视角风险
- 可复用写作资产

## 使用方式

先从对应集合的 `README.md` 和 `batch_summaries/` 建立整体地图，再进入 `papers/` 阅读单篇深度拆解。若目标是学习写法，优先看每篇的章节结构、文笔画像、常用词句和引用策略；若目标是找选题与创新套路，优先看选题层、gap 制造、创新性判断和论证链条。
